# Course Forge Prototype

Premium React/Vite front-end prototype για AI course creator dashboard.

## Run

```bash
npm install
npm run dev
```

Άνοιγμα στο browser:

```bash
http://localhost:8000
```

## Περιλαμβάνει

- Dashboard για course modules
- AI production panel
- Activity factory
- Export preview
- Responsive layout
- Premium SaaS αισθητική

## Επόμενα βήματα

- Σύνδεση με πραγματικό backend
- Upload Word/PDF/assets
- Course JSON schema
- Export standalone HTML/Moodle package
- User accounts και progress tracking
